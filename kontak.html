Sering Ditanyakan (FAQ)

Jawaban cepat untuk pertanyaan seputar program KUMARA.

Apakah aplikasi ini gratis untuk UMKM? +

Ya, aplikasi KUMARA diberikan 100% GRATIS kepada mitra UMKM binaan di Desa Ciburayut selama program berlangsung, termasuk pendampingan penggunaannya.

HP apa yang dibutuhkan untuk menggunakan KUMARA? +

Aplikasi KUMARA sangat ringan (Lite Version). Bisa dijalankan di smartphone Android entry-level dengan RAM minimal 2GB dan sistem operasi Android 8 ke atas.

Bagaimana jika data pelanggan saya hilang? +

Jangan khawatir! Data tersimpan aman di sistem Cloud (server) kami secara otomatis. Jika HP Anda rusak atau ganti HP baru, data akan kembali setelah login.

Apakah program ini akan berlanjut tahun depan? +

Target kami adalah keberlanjutan (Sustainability). Setelah fase inisiasi Innovillage selesai, pengelolaan teknis akan diserahterimakan secara bertahap kepada unit digital BUMDes Ciburayut.
